Sobre mim

Formado como Técnico em Informática pelo IFMA, trabalho hoje como desenvolvedor web utilizando WordPress. Na agência onde trabalho crio todo tipo de site, tanto institucional como loja virtual.

Meu primeiro projeto foi feito em equipe com alunos do 3° ano do curso Técnico em Informática do IFMA. Nesse projeto, criamos um site para o maior evento Geek de Açailândia - MA, atuei como desenvolvedor web e coordenador do projeto.

Depois disso, consegui meu primeiro emprego como desenvolvedor web, na Web Cloud Agency, onde estou atuando no mercado de trabalho há 3 meses, tendo já feito 5 leadpages e 5 sites institucionais, junto com um design UI.
